Word Count
----------

## DESCRIPTION

This library allows you to pass in any text that will be parsed and each word will be counted for the number of occurances.

Here are some guidelines:

1. The word count is case insensitive. "Hello" is the same as "hello"
1. It supports contractions
1. It supports time (e.g. "3:16")
1. It supports time periods (e.g. "a.m.")

## RUNNING

To run the following, you need:

1. Ruby 2.1.4
1. The `Bundler` gem installed

### Run tests

1. Install the dependencies `bundle install`
1. Run the test suite `bundle exec rspec spec`

### Run with the specified output

1. Install the dependencies `bundle install`
1. Run the `Rake` command `bundle exec rake count_words`
1. You can also run it against additional files with `bundle exec rake "count_words[file/path]"`
